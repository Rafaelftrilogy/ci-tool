How CI Works

What is this tool?

The single source of truth for Trilogy Care's Continuous Improvement register. Every improvement — whether identified internally, from a complaint, from the Commission, or through an audit — lives here with a unique reference ID.

Reference IDs

Every item gets a permanent ID: TC-CI-YYYY-NNNN (e.g. TC-CI-2026-0001). This traces back to the source system where it was identified.

Evaluation Gate

New submissions go to "Pending Review". A Manager or Admin reviews and either approves it as new, links it to an existing CI item (avoiding duplicates), or rejects it with a reason.

Standards

Every CI item must be linked to at least one Strengthened Aged Care Quality Standard (1–5). Items without a standard mapping are flagged for review.

Data Sources

The tool pulls from: Zoho CRM, Linear, Confluence, M365 Email (Commission), Complaints, and manual entries. Click "⟳ Refresh" to check for updates.

Statuses
