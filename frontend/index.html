📖 Help & Guide

What is this?

The single source of truth for Trilogy Care's Continuous Improvement. Every improvement — from complaints, audits, self-identified, or the Commission — lives here with a unique reference ID (TC-CI-YYYY-NNNN).

How to add a new item

Click + New Entry in the sidebar or toolbar. Fill in the mandatory fields. Your submission goes to the Evaluation Gate for review before it enters the register.

Evaluation Gate

A Manager/Admin reviews pending items and can:
• Approve — becomes a new CI item
• Link — attach to an existing item (avoids duplicates)
• Reject — with a logged reason

Standards Mapping

Every item should be linked to at least one Strengthened Standard (1–5). Unmapped items are flagged with ⚠. Go to Standards Map to fix them.

Data Sources

Items come from: Zoho CRM, Linear, Confluence, M365 Email, Complaints, SIRS, or manual entry. Each item tracks where it came from. Click Refresh Data to pull updates.

Statuses

Pending Review → awaiting gate
Identified → approved, awaiting action
In Progress → work underway
Implemented → done, needs evidence
Verified → evidence confirmed
Closed → complete

Reference IDs

Format: TC-CI-2026-0001. Generated automatically. Permanent. Traces back to source system.
